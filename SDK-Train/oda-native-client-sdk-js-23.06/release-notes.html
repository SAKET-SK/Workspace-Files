Oracle Digital Assistant Native Client SDK for Web

Release 23.06

We are thrilled to share the 23.06 release of the Oracle Digital Assistant Native Client SDK for Web. This time we are introducing editable forms and making Oracle TTS the default experience. We've also included a bug fix for TTS.

What's New

  • Editable Forms - The SDK now supports the rendering of editable forms, which accept user input through such fields as single-select and multi-select lists, time pickers, date pickers, and toggle switches. Refer to our guide for more details.

What's Improved

  • Accept the HTTPS protocol in the URI - You can now pass the first path of the Oracle Digital Assistant URI either with, or without, the protocol (https://).
  • Oracle TTS as the default TTS service - We've made Oracle Cloud Infrastructure (OCI) TTS the default TTS experience. Refer to our guides for more details.

What's Fixed

  • Text To Speech fixes - Previously, TTS was uttering all of the contents within the <script> and <style> tags and also hidden HTML elements. This has now been fixed.
