Oracle Digital Assistant Native Client SDK for Web

Release 23.08

We are thrilled to share the 23.08 release of the Oracle Digital Assistant Native Client SDK for Web. This time we are introducing advanced forms and lots of under-the-hood improvements. We've also included bug fixes for autocomplete, TTS, and embedded mode.

What's New

  • Advanced Forms - We now support advanced form layouts which let you create a custom message layout using rows and columns. The form message can now have a list of rows and each row can have multiple columns. A column can have a list of read only or editable fields. Refer to our guide for more details.

What's Improved

  • Render Read Only Fields in Table Messages - We now support the rendering of all types of read-only fields in Table messages, along with Form, Edit-Form, and Table-Form messages. Refer to our guides for more details.

What's Fixed

  • Autocomplete Suggestions - Previously, autocomplete suggestions did not get updated as the user typed. This has now been fixed.
  • Text to Speech - TTS was not speaking out the contents of the Table Form messages. This has now been fixed.
  • Embedded Mode - In the embedded mode, the chat widget had been positioned using 20 px for the right and bottom. Also, the share popup menu is displayed at the bottom right, irrespective of the widget position. These issues have now been fixed.
